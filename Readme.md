 Script 3: Directory & Disk Audit
Purpose:
Analyzes important system directories.
Features:
Checks permissions
Displays owner information
Shows directory size
Provides disk usage summary
Command to Run:
bash script3.sh
 Script 4: Log File Analyzer
Purpose:
Analyzes log files for specific keywords.
Features:
Counts total lines
Finds keyword matches (default: "error")
Calculates percentage of matches
Displays last 5 matching lines
Command to Run:
bash script4.sh <logfile> [keyword]
Example:
bash script4.sh syslog error
